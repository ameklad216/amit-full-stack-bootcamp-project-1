# FlowSync – SaaS Landing Page

A modern, fully responsive SaaS landing page built using **HTML5, CSS3, and Vanilla JavaScript**. This project was developed as **Project 1** for the ** AMIT-LEARNING Full Stack Freelancing Bootcamp** and demonstrates the ability to create a professional, conversion-focused landing page without using any CSS frameworks.

---

## 📖 Overview

FlowSync is a fictional Software-as-a-Service (SaaS) product designed to showcase a modern landing page with clean UI, responsive layouts, interactive components, and conversion-focused design.

The project follows semantic HTML practices, responsive design principles, and includes JavaScript-powered interactions.

---

## 🚀 Features

* Modern Hero Section
* Responsive Navigation Bar
* Mobile Hamburger Menu
* Features Section
* Testimonials Section
* Pricing Plans
* Interactive FAQ Accordion
* Contact Form
* Smooth Scrolling
* Scroll Reveal Animations
* Hover Effects
* Responsive Design for Mobile, Tablet, and Desktop

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/flowsync.git
```

2. Navigate to the project folder

```bash
cd flowsync
```

3. Open `index.html` in your browser.

No additional dependencies or installation steps are required.
